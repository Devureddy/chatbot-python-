# 💗 Python Chatbot

✨ My first mini project built using Python & Jupyter Notebook

---

## What this chatbot can do

* Responds to basic greetings like **hello**
* Answers simple questions like **"how are you"**
* Performs **basic math calculations** (+, -, *, /)
* Exits when user types **bye**

---

##  Tech Used

* Python 
* Jupyter Notebook 

---

##  How it works

This chatbot uses:

* `while loop` → keeps the chat running
* `if-elif conditions` → checks user input
* `input()` → takes user messages
* `eval()` → solves math expressions

---

## Example Interaction

You: hello
Chatbot: Hi there!

You: 5 + 3
Chatbot: 8

You: how are you
Chatbot: I'm just code, but I'm doing great!

You: bye
Chatbot: Goodbye!
